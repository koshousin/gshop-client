*思考* ：
   + 什么情况下在组件内直接请求，什么情况下需要 dispatch ?
   + vuex 的一套写作流程是什么？
     > 一般来讲
       + state -> mutation-types -> mutations() -> action(同步/异步操作)
       + 应用层级的状态应该集中到单个 store 对象中。
       + 提交 mutation 是更改状态的唯一方法，并且这个过程是同步的。
       + 异步逻辑都应该封装到 action 里面。
